Les évolutions du CSS

CSS Grid

3 days ago by Daganaud Alex

Le CSS Grid Layout est le système de mise en page le plus puissant disponible en CSS. C'est un système bidimensionnel, ce qui signifie qu'il peut gérer à la fois des colonnes et des lignes, contrairement à flexbox qui est en grande partie un système unidimensionnel. Vous travaillez avec la grille en appliquant des règles CSS à la fois à un élément parent (qui devient le conteneur de la grille) et aux enfants de cet élément (qui deviennent les éléments de la grille).

Flexbox

Étant donné que flexbox est un module entier et non une propriété unique, il implique beaucoup de choses, y compris l'ensemble de ses propriétés. Certaines d'entre elles sont destinées à être placées sur le conteneur (élément parent, connu sous le nom de "conteneur flex") tandis que les autres sont destinées à être placées sur les enfants (dites "éléments flex").

Les features queries

En programmation classique, il faut des variables et des structures conditionnelles pour commencer à s'amuser un peu, et CSS l'a bien compris. C'est pour cette raison que les features queries (règle @supports { ... }) entrent en jeu. Il ne s'agit pas à proprement parler de if/else comme dans la programmation habituelle, mais le résultat en est presque identique malgré tout. En effet, @supports permet de détecter si une propriété ou une fonctionnalité existe et fonctionne dans le navigateur en cours d'utilisation.
